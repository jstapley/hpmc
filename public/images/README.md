# Images Folder

Place your images here following this structure:

## Hero
- `hero-bg.jpg` — Full-screen hero background (recommended: 1920x1080px or larger)

## Client Logos (in /clients/)
- `outdoor-world.png`
- `hermitage-bay.png`
- `blue-waters.png`
- `barefoot-antigua.png`

## Portfolio Work Photos (in /work/)
- `work-1.jpg` through `work-6.jpg`
- Recommended: landscape orientation, 1200x800px minimum

All images should be optimized for web (use tools like https://squoosh.app or https://tinypng.com).
